# WWII Battles and Weapons Website

A simple web application showcasing major battles and weapons from World War II, built with Node.js, Express, and EJS.

## Features

- **Multi-language support**: English, Hungarian, and Romanian
- **Battles section**: Information on key WWII battles
- **Weapons section**: Details on famous WWII weapons
- **Quiz**: Test your knowledge with a simple quiz
- **Responsive design**: Basic CSS styling

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kilencje/tori.git
   cd tori
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser and go to `http://localhost:3000`

## Usage

- Navigate through battles and weapons
- Switch languages using the query parameter `?lang=en`, `?lang=hu`, or `?lang=ro`
- Take the quiz at `/quiz`

## Technologies Used

- Node.js
- Express.js
- EJS templating
- Vanilla JavaScript for quiz functionality
